# certamen 1 - Benjamin Colicoi Lizama

## Descripción
certamen web personal desarrollado como evaluación práctica para el módulo de Desarrollo Web Front-End. El sitio es una Single Page Application (SPA) responsiva que presenta mi perfil profesional como estudiante de Analista Programador en INACAP.

## Tecnologías Utilizadas
- **HTML5:** Estructura semántica completa.
- **CSS3:** Maquetación con Flexbox y CSS Grid.
- **Diseño Responsivo:** Media Queries para adaptación a dispositivos móviles y tablets.
- **Git y GitHub:** Control de versiones con commits semánticos.

## Cómo ejecutar el proyecto
1. Clonar este repositorio usando la terminal:
   `git clone https://github.com/manologithu/certamen_1_benja`
2. Navegar a la carpeta del proyecto.
3. Abrir el archivo `index.html` en cualquier navegador web moderno (se recomienda Google Chrome o Mozilla Firefox).
